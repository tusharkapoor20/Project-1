# Analysis of Popular Games (1980-2023)
## Overview
This repository contains the code and analysis for a study of popular games spanning from 1980 to 2023. The analysis explores various aspects of gaming trends, including sales, genres, platforms, and more.

## Dataset
The analysis is based on a comprehensive dataset of popular games collected from various sources. The dataset includes information such as game title, release year, genre, platform, sales figures, and critical reception.

## Analysis
The analysis covers the following aspects:

Trends in game releases over time
Distribution of game genres
Dominant gaming platforms
Sales performance across different regions
Correlation between critical reception and sales
## Usage
To reproduce the analysis, follow these steps:

Clone this repository to your local machine.
Navigate to the project directory.
Install the required dependencies (if any).
Run the main analysis script.

## Dependencies
Python 3.x, Pandas, Matplotlib, Seaborn
